# StonksOverFlow
## Убийца TradingView
Какую то гробовую тему выбрал. 
Проект представляет собой платформу для технического анализа финансовых инструментов. 

### Для регистрации пользователя требуется 
- email
- Ник
- пароль

### Права пользователей
 - Неавторизованный пользователь может смотреть графики, но не может добавлять в избранные инструменты для себя.

 - Авторизованный пользователь может составить себе портфель из интересных ему инструментов. Удалять тоже может.
- Предполагается,что инструменты могут добавляться только админом, во имя избежания добавления разного сорта мемкоинов обычными пользователями.
 К каждому инструменту Админ должен будет добавлять описание инструмента.
### Непосредственно сам проект
- Предполагается,что обновление рыночных данных будет происходить в реальном времени.(возможно с задержкой,хз как там по бесплатным апи вопрос обстоит,да и не торговый терминал пишем,так что не критично)

- Предполагается наличие некоторых технических индикаторов(будут только рабочие и самые простые(MA,EMA).
 Возможно реализуется коррекция по Фибоначчи(да не возможно,а 99%)
- У пользователей будет возможность рисовать стрелки, линии,прямоугольники на графике. 
- Должна быть реализована группировка инструментов по классам для удобного поиска(индексы, акции, фьючерсы и т.д)

- У инструмента должна быть также динамика за день, месяц и год(% изменения).
- Будет доступно 8 таймфреймов - [5 минут,15 минут, 30 минут,1 час,4 часа,1 день,1 неделя,1 месяц]
- Вид графика будет только один - в виде японских свечей.
